# rn-news-app
news feedを取得して表示するアプリ  
expoを利用した素振りとして

# setup
1. [expo cli](https://docs.expo.dev/workflow/expo-cli/)のインストール
2. [news api](https://newsapi.org/)のアカウント作成-> `.env`にNEWS_API_KEYを設定
3. `yarn start`で起動
